# Manufacturing Operations Analytics Dashboard

## Project Overview
Developed an interactive Power BI dashboard to analyze manufacturing operations for Aeronex Precision Components Pvt. Ltd.

## Objective
The dashboard helps management monitor:

- Production Performance
- Machine Utilization
- Downtime Analysis
- Quality Inspection
- Overall Equipment Effectiveness (OEE)

## Tools Used

- Power BI
- Power Query
- DAX
- Excel

## Data Modeling

- Star Schema
- Fact & Dimension Tables
- Relationships

## KPIs

- Total Production
- Production Achievement %
- Machine Utilization %
- Downtime %
- Defect Rate %
- Overall Equipment Effectiveness (OEE)

## Dashboard Features

- Executive KPI Cards
- Plant-wise Analysis
- Machine Performance
- Operator Performance
- Shift Analysis
- Product Category Analysis
- Downtime Trend Analysis
- Quality Analysis

## Skills Demonstrated

- Data Cleaning
